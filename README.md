# Annotated SBGNML Generator

This code can generate randomly generated, but valid SBGN PD diagrams and (optionally) annotate them. This could be used to a create a large collection of consistently sized and annotated SBGN diagrams.

# Run
```
uv run sbgnml_annotated_generator.py --output-dir annotated
```

# Uses
* SBGNML generator: https://sciluna.github.io/image-to-sbgn-analysis/dataset/index.html
* Identifier service: https://grounding.indra.bio/
* playwright for browser automation
